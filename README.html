T7 Frontend Boilerplate

T7 Frontend Boilerplate helps you build prototypes and web application UI's, fast.

Setup

Step 1 Install NodeJS

Simply use the installer for initial installation and updates.

Step 2 Checkout GIT Repository

GIT git@git.assembla.com:next-gen-collaboration-model.git

HTTP https://git.assembla.com/next-gen-collaboration-model.git

Step 3 Get npm packages

Go to the directory of the repository you just checkout and run:

> npm install

This will install the follow node dependencies in /node_moduels:

"devDependencies": {
    "grunt": "latest",
    "grunt-contrib-requirejs": "latest",
    "connect": "latest",
    "bower": "latest"
  }
Step 4 Get bower components

Go to the directory of the repository you just checkout and run:

> bower install
